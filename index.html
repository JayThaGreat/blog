A quick bio - I'm Jelson Rodriguez and I've taken on the challenge of going through Learn Verified's Full Stack Web Development program at The Flatiron School. Already having some background in Java, javascript and HTML; I knew the program would be a fun challenge. After going through a portion of the course, one of my first projects involved building a gem that interacts with the user via the command line interface(CLI). I built a tool that can scrape Mitre's website which contains repositories of Common Vulnerabilities and Exposures (CVEs). Mitre also describes these CVEs as publicly known cybersecurity vulnerabilities. This list is useful to determine the vulnerabilities that affect different types of internet protocols, software and hardware.
